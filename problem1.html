b tag

The tag in HTML is used to specify the bold text without any extra importance. The text is written within tag display in bold size. You can do that by using font-weight: bold; property in CSS. It is a container tag that contains an opening tag, content & closing tag

i tag

The i tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. u tag

u The Unarticulated Annotation (Underline) element. The u HTML element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation

select tag

The select element is used to create a drop-down list. The select element is most often used in a form, to collect user input.

button tag

The button tag defines a clickable button. Inside a button element you can put text (and tags like , b, strong , br, img , etc.)
